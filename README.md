
# Cosmetics Product Management System (CPMS) 💄

## 💡 About Our Project
This is our project for the **Software Engineering Lab**. We are building a system to help cosmetics shops manage their products and suppliers easily. Instead of using paper or messy Excel files, our system (CPMS) makes everything organized and safe .

## ❓ Why we chose this? (The Problem)
In cosmetics shops, there are too many products (skincare, makeup, hair tools). Currently, it is hard to:
* Link products to their specific suppliers .
* Avoid mistakes in prices or deleting items by accident .
* Keep data safe because anyone can change anything .

##  What the system does (Features)
We are working on these main features:
* **Managing Products:** Add, Edit, and Delete cosmetics items .
* **Categories:** Sorting products into (Makeup, Skincare, etc.) .
* **Suppliers:** Linking each product to its supplier .
* **Dashboard:** A simple page to see how many products we have in total .
* **Security:** Login system so only the manager can delete important data .

##  Tools we are using
* **Framework:** Laravel (PHP) .
* **Database:** MySQL .
* **Design:** HTML, CSS, and Bootstrap .
* **Organization:** GitHub for code and we are learning **Agile** .
##  How we work (Agile Methodology)
Following Eng. Belal's instructions, we are working step-by-step using **Sprints** .
* We focus on **System Analysis** first (Requirements and User Stories) .
* We build the project part by part to make sure everything is correct .
